# 抑郁风险预警系统

基于CatBoost模型的社区中老年人抑郁风险筛查工具

## 项目简介

本系统基于中国健康与养老追踪调查(CHARLS)数据构建的CatBoost机器学习模型，专门用于社区中老年人的抑郁风险筛查。系统通过13项简易健康指标，快速评估个体抑郁风险，并提供个性化的健康建议。

## 功能特点

- **科学评估**：基于CatBoost机器学习模型，AUC达0.7805
- **快速筛查**：仅需2-3分钟完成评估
- **解释透明**：提供SHAP可解释性分析，展示影响因素
- **个性化建议**：根据风险等级提供针对性建议
- **历史记录**：保存评估历史，支持导出和管理
- **PDF报告**：生成专业的评估报告

## 技术架构

- **前端**：Streamlit Web应用
- **后端**：Python
- **模型**：CatBoost分类器
- **存储**：SQLite本地数据库
- **依赖管理**：pip

## 项目结构

```
depression_risk_app/
├── app.py                    # 主程序
```

## 安装指南

### 1. 克隆项目

```bash
git clone <repository_url>
cd depression_risk_app
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 准备模型文件

确保 `models/` 目录下存在 `CatBoost_F2_v1_with_threshold.pkl` 模型文件。

## 运行应用

```bash
streamlit run app.py
```

应用将在浏览器中打开，默认地址为 `http://localhost:8501`。

## 使用说明

### 1. 风险评估

1. **填写健康信息**：按照页面提示填写基本信息、健康状况、功能状态、生活方式和慢性病管理等信息
2. **使用速测工具**：点击ADL/IADL速测工具辅助填写功能状态评分
3. **开始评估**：点击"开始评估"按钮进行分析
4. **查看结果**：系统将显示风险评估结果、影响因素分析和个性化建议
5. **保存结果**：可添加备注并保存评估结果到历史记录

### 2. 历史记录

- **查看记录**：浏览所有评估历史
- **筛选记录**：按时间范围、风险等级和备注搜索
- **导出数据**：导出CSV格式的历史记录
- **删除记录**：支持单条删除和批量清空
- **生成报告**：为每条记录生成PDF格式的评估报告

### 3. 使用示例

系统提供了高风险和低风险的示例数据，点击侧边栏的"加载高风险示例"或"加载低风险示例"按钮即可快速体验。

## 数据隐私

- 所有评估均为匿名
- 不收集个人身份信息
- 数据仅存储在本地SQLite数据库
- 评估结果可随时删除

## 医疗声明

本系统仅为抑郁风险初步筛查工具，不替代专业医疗诊断。评估结果仅供参考，如有身体不适或情绪困扰，请及时前往社区卫生服务中心或心理科进一步检查。

## 故障排除

### 常见问题

1. **模型文件未找到**：确保 `models/CatBoost_F2_v1_with_threshold.pkl` 文件存在
2. **依赖项安装失败**：尝试使用 `pip install --upgrade pip` 升级pip后重新安装
3. **PDF报告生成失败**：确保已安装 `reportlab` 库
4. **数据库连接错误**：系统会自动创建数据库文件，无需手动配置

### 日志和调试

应用运行过程中的错误信息会显示在Streamlit界面上，便于排查问题。
